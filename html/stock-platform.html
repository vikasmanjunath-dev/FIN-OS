INTELLIGENCE // 03

The Brain — Rule Engine, visualised

The Smart Insights aren't black-magic ML. Every signal is a transparent, auditable rule.

RSI Overbought / Oversold
0–30 oversold 30–70 neutral 70–100 overbought
if rsi > 70 → "Overbought, fade strength" if rsi < 30 → "Oversold, watch for reversal"
MA Trend Regime
if price > MA200 and MA50 > MA200 → "Strong uptrend" if price < MA200 and MA50 < MA200 → "Bearish regime"
VOL Unusual Volume
z = (vol - mean20) / std20 if z > 2 → "Volume spike — institutional flow?"
52W Range Proximity
52W Low52W High
pct = (price - low52) / (high52 - low52) if pct > 0.95 → "Trading near 52-week high"

API // 05

Public Contract

Twelve clean endpoints. Fully typed. CORS-ready. Click to copy curl.

GET /api/stock/{symbol}

Spot quote + market cap, PE, 52W, volume.

GET /api/history/{symbol}

OHLCV. Intervals: 1m → 1mo. Range: 1d → max.

GET /api/indicators/{symbol}

RSI, MACD, MA20/50/200, Bollinger, ATR.

GET /api/insights/{symbol}

Rule-engine signals + plain-English narrative.

GET /api/depth/{symbol}

Simulated 5-level bid/ask book.

GET /api/fundamentals/{symbol}

EPS, P/E, P/B, ROE, debt, margins.

GET /api/sectors

NSE sector indices for the heatmap.

GET /api/search?q=

Fuzzy ticker autocomplete.

WS /ws/ticks

WebSocket — live tick stream (3-5s).
